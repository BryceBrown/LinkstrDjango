Documentation


How to add a user to your company
Simple! Go to the company settings page (You have to be an admin for your company to do this) and click the User Invites: button! Simply fill out an email address
What is an interstitial?
An interstitial is a page that is shown before a user is redirected to a link that you have
How to delete users
Sometimes, breakups happen. Simply navigate to the setting screen, select the users tab, click on the user you wish to remove, and click the delete button. This will delete the selected user account. The owner of the company cannot be deleted.
Add/Edit a domain
In order to generate links on a domain that you own, you must first add the domain to our system.
Remove a domain
In order to remove a domain, simply select the domain from the list and click the delete button from the dialog that appears. Use this feature with caution, as when removed, all links associated with a domain will no longer function.
Switch active domains
In order to switch active domains, click the domain on the right that is currently active. This should bring up a modal that will allow you to switch domains.
Generate a link
In order to collect statistics, you must first generate some links! First, navigate to the dashboard. From here, simply paste a valid url into the Shorten URL field and click Go. Thats it! Once the url is generated, a dialog should appear to let you copy and paste it wherever you like. There are also several social media share buttons for your convenience.
Decommission a link
Sometimes, you no longer wish to see and monitor links in your dashboard. To remove a link from your list of generated links, simply select the link from the left side list to make it active, and press the Decommission button. Clicks on this link will still work, but you will no longer see it in your list of generated links.
View Deactivated links
This feature is not yet available.
View statistics on a generated link
First, make sure that you have the domain that the link was generated under. Then, simply find through paging or search for the link, and click on it! We have data from where the link was collected, What browser it was collected on, and how many times it was clicked each day.
Generate a customized link
On the Dashboard page, click the Customize button. Then, simply enter the URL you wish to use and the link you wish it to point to.
